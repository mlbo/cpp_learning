## 🎯 本周学习重点

- 建立算法复杂度意识：能分析 O(1)、O(log n)、O(n)、O(n log n) 的常见场景
- 掌握数组题核心模板：快慢指针、左右指针、逆向双指针
- 理解 C++11 基础语法：`auto`、`decltype`、`nullptr`、`constexpr`、统一初始化
- 熟练应用两类高频算法：滑动窗口和二分查找
