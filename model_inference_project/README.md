# C++ Model Inference Project

A complete C++ model inference implementation demonstrating multi-threaded neural network inference.

## Features

- **Custom Tensor Library**: Multi-dimensional array with math operations
- **Neural Network Layers**: Linear, ReLU, Sigmoid, Softmax, Dropout, BatchNorm, LayerNorm
- **Sequential Model**: Chain layers together for easy model building
- **Multi-threaded Inference Engine**: Parallel batch inference with thread pool
- **Async Inference**: Non-blocking inference with futures
- **Performance Benchmarking**: Throughput and latency measurement

## Requirements

- **Compiler**: GCC 4.8+ or Clang 3.4+ (C++11 support)
- **CMake**: Version 3.10+
- **OS**: Linux / macOS / Windows (with MinGW)

## Usage Examples

### Creating and Using Tensors

```cpp
#include "tensor.h"
using namespace inference;

// Create tensors
auto t1 = Tensor::random({3, 4});  // Random 3x4 matrix
auto t2 = Tensor::zeros({10});     // Zero vector

// Operations
auto sum = t1 + t2;                // Element-wise add
auto product = t1.matmul(t2);      // Matrix multiplication
auto activated = t1.relu();        // ReLU activation
auto probs = t1.softmax();         // Softmax
```

### Building a Neural Network

```cpp
#include "layers.h"
using namespace inference;

// Create MLP model
Sequential model;
model.add(Linear(784, 256));   // Input layer
model.add(ReLU());
model.add(Linear(256, 128));   // Hidden layer
model.add(ReLU());
model.add(Linear(128, 10));    // Output layer
model.add(Softmax());

// Print model structure
model.printStructure();

// Forward pass
Tensor input = Tensor::random({784});
Tensor output = model.forward(input);
```

### Multi-threaded Inference

```cpp
#include "inference_engine.h"
using namespace inference;

// Create model and engine
Sequential model = createMLPClassifier();
ParallelInferenceEngine engine(&model, 4);  // 4 threads

// Async inference
auto future = engine.inferAsync(input);
InferenceResult result = future.get();

// Batch parallel inference
std::vector<Tensor> inputs = {...};
auto results = engine.inferBatchParallel(inputs);
```

## Performance

Typical throughput on a modern multi-core CPU:
- Single-threaded: ~100-500 samples/sec
- 4 threads: ~300-1500 samples/sec
- 8 threads: ~500-2500 samples/sec

Actual performance depends on model size, batch size, and hardware.
