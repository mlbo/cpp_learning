# LeetCode 283: 移动零

## 题目描述

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的**相对顺序**。

### 要求
- 必须在原数组上操作，不能拷贝额外的数组
- 尽量减少操作次数

### 示例

```
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```

---

## 解题思路

### 方法一：双指针交换法（推荐）

#### 核心思想
- **快指针 (fast)**：遍历数组，寻找非零元素
- **慢指针 (slow)**：记录下一个非零元素应该放置的位置

#### 算法流程

```
初始化: slow = 0

遍历 (fast = 0 到 n-1):
  如果 nums[fast] != 0:
    交换 nums[slow] 和 nums[fast]
    slow++
```

#### 为什么能保持相对顺序？
因为我们是按顺序找到非零元素，并按顺序放到前面的位置。

### 方法二：两次遍历法

#### 思路
1. 第一次遍历：将非零元素依次移动到数组前面
2. 第二次遍历：将剩余位置填充零

### 方法三：优化版

#### 优化点
当 `slow == fast` 时，不需要交换（元素已在正确位置）

---

## 代码实现

```cpp
void moveZeroes(vector<int>& nums) {
    int slow = 0;  // 下一个非零元素的位置

    for (int fast = 0; fast < nums.size(); ++fast) {
        if (nums[fast] != 0) {
            swap(nums[slow], nums[fast]);
            ++slow;
        }
    }
}
```

## 图解示例

### 示例：nums = [0, 1, 0, 3, 12]

```
初始状态:
nums: [0, 1, 0, 3, 12]
       s  f
(s = slow, f = fast)

步骤 1: nums[1] = 1 ≠ 0
       交换 nums[0] 和 nums[1]
nums: [1, 0, 0, 3, 12]
          s  f

步骤 2: nums[2] = 0
       跳过
nums: [1, 0, 0, 3, 12]
          s     f

步骤 3: nums[3] = 3 ≠ 0
       交换 nums[1] 和 nums[3]
nums: [1, 3, 0, 0, 12]
             s     f

步骤 4: nums[4] = 12 ≠ 0
       交换 nums[2] 和 nums[4]
nums: [1, 3, 12, 0, 0]
                s     f

结果: [1, 3, 12, 0, 0]
```

## 算法分析

| 方法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|-----------|-----------|------|
| 双指针交换 | O(n) | O(1) | 一次遍历，代码简洁 |
| 两次遍历 | O(n) | O(1) | 逻辑清晰，减少交换 |
| 优化版 | O(n) | O(1) | 减少自交换 |

## 关键洞察

### 为什么双指针有效？

1. **分区思想**：将数组分为"非零区"和"待处理区"
2. **一次遍历**：每个元素最多被交换一次
3. **保持顺序**：非零元素按出现顺序放入

### 与其他问题的联系

这个问题是"数组分区"问题的一种特化形式：
- 快速排序的 partition：以枢轴为界分区
- 本题：以零为界分区，零放到末尾

### 变体 1：移动偶数到末尾

```cpp
void moveEvens(vector<int>& nums) {
    int slow = 0;
    for (int fast = 0; fast < nums.size(); ++fast) {
        if (nums[fast] % 2 != 0) {  // 奇数
            swap(nums[slow], nums[fast]);
            ++slow;
        }
    }
}
```
