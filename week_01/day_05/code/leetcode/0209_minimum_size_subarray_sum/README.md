# LeetCode 209: 长度最小的子数组

## 题目描述

给定一个含有 `n` 个正整数的数组和一个正整数 `target`。

找出该数组中满足其和 `≥ target` 的长度最小的 **连续子数组**，并返回其长度。如果不存在符合条件的子数组，返回 `0`。

## 示例

**示例 1:**
```
输入: target = 7, nums = [2,3,1,2,4,3]
输出: 2
解释: 子数组 [4,3] 是该条件下的长度最小的子数组。
```

**示例 2:**
```
输入: target = 4, nums = [1,4,4]
输出: 1
```

**示例 3:**
```
输入: target = 11, nums = [1,1,1,1,1,1,1,1]
输出: 0
```

## 解题思路

### 方法一：滑动窗口（推荐）

```
┌─────────────────────────────────────────────────────────────┐
│                     滑动窗口算法流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  初始化: left = 0, sum = 0, minLen = ∞                      │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  for right in 0..n-1:                               │   │
│  │      sum += nums[right]    // 扩展窗口               │   │
│  │                                                     │   │
│  │      while sum >= target:  // 满足条件              │   │
│  │          minLen = min(minLen, right - left + 1)     │   │
│  │          sum -= nums[left++]  // 收缩窗口            │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  返回: minLen == ∞ ? 0 : minLen                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**滑动窗口正确性证明：**

1. **单调性**：由于数组元素全为正数，窗口扩展时sum增大，收缩时sum减小
2. **完备性**：所有可能的子数组都会被遍历到
3. **最优性**：每次收缩都保证了最小长度

### 方法二：前缀和 + 二分查找

```
步骤：
1. 计算前缀和数组 prefix[n+1]
2. 对于每个起始位置 i，二分查找满足 prefix[j] - prefix[i] >= target 的最小 j
3. 更新最小长度

时间: O(n log n)    空间: O(n)
```

### 方法三：暴力法

```
枚举所有子数组，计算和

时间: O(n²)    空间: O(1)
```

## 代码实现

```cpp
int minSubArrayLen(int target, vector<int>& nums) {
    int left = 0, sum = 0, minLen = INT_MAX;
    
    for (int right = 0; right < nums.size(); ++right) {
        sum += nums[right];  // 扩展窗口
        
        while (sum >= target) {  // 收缩条件
            minLen = min(minLen, right - left + 1);
            sum -= nums[left++];
        }
    }
    
    return minLen == INT_MAX ? 0 : minLen;
}
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 滑动窗口 | O(n) | O(1) |
| 前缀和+二分 | O(n log n) | O(n) |
| 暴力法 | O(n²) | O(1) |

## 关键点

1. **滑动窗口适用条件**：
   - 数组元素全为正数
   - 求满足条件的连续子数组
   - 问题具有单调性

2. **边界情况**：
   - 单个元素满足条件
   - 所有元素和不满足条件
   - 空数组

3. **优化技巧**：
   - 使用 `INT_MAX` 初始化最小值
   - 及时收缩窗口，避免漏解
