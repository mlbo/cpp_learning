# LeetCode 61: 旋转链表

## 题目描述

给定链表的头节点 `head`，旋转链表，将链表每个节点向右移动 `k` 个位置。

## 示例

### 示例 1
```
输入：head = [1,2,3,4,5], k = 2
输出：[4,5,1,2,3]
```

图示：
```
原链表: 1 -> 2 -> 3 -> 4 -> 5
旋转后: 4 -> 5 -> 1 -> 2 -> 3
```

### 示例 2
```
输入：head = [0,1,2], k = 4
输出：[2,0,1]
```

解释：
- k = 4, n = 3
- 有效旋转次数 = 4 % 3 = 1
- 向右旋转1次得到 [2, 0, 1]

## 解题思路

### 方法一：成环后断开（推荐）

**思路**：
1. 计算链表长度
2. 将链表首尾相连形成环
3. 找到新的断开点
4. 断开并返回新头节点

**复杂度分析**：
- 时间复杂度：O(N)，遍历链表
- 空间复杂度：O(1)，常数空间

**代码**：
```cpp
ListNode* rotateRight(ListNode* head, int k) {
    if (!head || !head->next || k == 0) return head;

    // 1. 计算长度并找到尾节点
    int len = 1;
    ListNode* tail = head;
    while (tail->next) {
        tail = tail->next;
        len++;
    }

    // 2. 计算有效旋转次数
    k = k % len;
    if (k == 0) return head;

    // 3. 成环
    tail->next = head;

    // 4. 找到新的断开点
    int steps = len - k;
    ListNode* newTail = head;
    for (int i = 1; i < steps; i++) {
        newTail = newTail->next;
    }

    // 5. 断开并返回
    ListNode* newHead = newTail->next;
    newTail->next = nullptr;

    return newHead;
}
```

### 方法二：快慢指针

**思路**：
1. 计算链表长度
2. 快指针先走 k 步
3. 快慢指针同时走
4. 慢指针到达新尾节点时断开

**复杂度分析**：
- 时间复杂度：O(N)
- 空间复杂度：O(1)

### 方法三：两次遍历

**思路**：
1. 第一次遍历计算长度
2. 第二次遍历找到断开点

**复杂度分析**：
- 时间复杂度：O(N)
- 空间复杂度：O(1)

## 图解

### 旋转过程图解

```
原始链表:
    1 -> 2 -> 3 -> 4 -> 5

步骤1: 计算长度 n = 5
步骤2: k = 2 % 5 = 2
步骤3: 成环
    1 -> 2 -> 3 -> 4 -> 5
    ^                   |
    |-------------------|

步骤4: 找新断开点 (移动 n-k = 3 步)
    当前位置在节点3（新尾节点）
    新头节点 = 节点3的next = 节点4

步骤5: 断开
    4 -> 5 -> 1 -> 2 -> 3 -> nullptr
```

## 关键点

1. **k 可能大于链表长度**：需要取模 `k = k % n`
2. **成环技巧**：将尾节点的 next 指向头节点
3. **新断开点**：从头节点移动 `n - k` 步到达新尾节点
4. **边界处理**：空链表、单节点、k=0 的情况

## 方法比较

| 方法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|-----------|-----------|------|

| 快慢指针 | O(N) | O(1) | 思路清晰 |
| 两次遍历 | O(N) | O(1) | 直观易懂 |
