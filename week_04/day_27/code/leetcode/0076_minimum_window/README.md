# LeetCode 76: 最小覆盖子串

## 题目描述

给你一个字符串 `s` 和一个字符串 `t`。返回 `s` 中涵盖 `t` 所有字符（包括重复字符）的最小子串。如果不存在这样的子串，返回空字符串。

**示例：**
```
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小窗口子串 "BANC" 包含了 t 中的所有字符 'A'、'B'、'C'
```

## 形象化理解

想象你是一个快递员，需要收集一整套特定的物品（字符串t中的所有字符）。你在一条街道上行走（遍历字符串s），街道两边的店铺出售各种物品。

```
街道: A D O B E C O D E B A N C
      ├───────────────────┤     第一次找到ABC
      ADOBEC (长度6)
      
街道: A D O B E C O D E B A N C
                      ├───┤     最终找到最短的
                      BANC (长度4)
```

## 解题思路

### 方法：滑动窗口

这是一道典型的滑动窗口问题，使用双指针维护一个可变长度的窗口。

**核心步骤：**

1. **初始化**：统计目标字符串t中每个字符的数量需求
2. **扩展窗口**：右指针向右移动，将字符加入窗口
3. **判断条件**：当窗口包含所有需要的字符时，尝试收缩
4. **收缩窗口**：左指针向右移动，寻找更短的满足条件的窗口
5. **更新结果**：记录过程中的最小窗口

### 算法流程图

```
┌─────────────────────────────────────────┐
│           初始化字符计数                 │
│    need[] 统计t中各字符需求量           │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│         扩展窗口 (right++)              │
│    加入s[right]，更新window计数         │
└─────────────────┬───────────────────────┘
                  │
                  ▼
         ┌───────────────┐
         │ 窗口满足条件? │
         │(valid==need)  │
         └───────┬───────┘
                 │
        ┌────────┴────────┐
        │ 是              │ 否
        ▼                 │
┌───────────────┐         │
│ 更新最小窗口   │         │
└───────┬───────┘         │
        │                 │
        ▼                 │
┌───────────────┐         │
│ 收缩窗口       │         │
│ (left++)      │         │
└───────┬───────┘         │
        │                 │
        └────────┬────────┘
                 │
                 ▼
         ┌───────────────┐
         │ right遍历完?  │──否──► 继续扩展
         └───────┬───────┘

                 ▼
         ┌───────────────┐
         │   返回结果    │
         └───────────────┘
```

## 代码实现

```cpp
string minWindow(string s, string t) {
    // 边界条件
    if (s.empty() || t.empty() || s.size() < t.size()) {
        return "";
    }
    
    // 统计t中各字符需求
    vector<int> need(128, 0);
    for (char c : t) need[c]++;
    
    // 计算需要满足的字符种类数
    int needCount = 0;
    for (int i = 0; i < 128; i++) {
        if (need[i] > 0) needCount++;
    }
    
    // 滑动窗口
    vector<int> window(128, 0);
    int left = 0, right = 0;
    int valid = 0;
    int minLen = INT_MAX, minStart = 0;
    
    while (right < s.size()) {
        // 扩展窗口
        char c = s[right++];
        if (need[c] > 0) {
            window[c]++;
            if (window[c] == need[c]) valid++;
        }
        
        // 收缩窗口
        while (valid == needCount) {
            // 更新结果
            if (right - left < minLen) {
                minLen = right - left;
                minStart = left;
            }
            
            char d = s[left++];
            if (need[d] > 0) {
                if (window[d] == need[d]) valid--;
                window[d]--;
            }
        }
    }
    
    return minLen == INT_MAX ? "" : s.substr(minStart, minLen);
}
```

## 复杂度分析

- **时间复杂度**：O(n)，其中n是字符串s的长度。每个字符最多被访问两次（一次被right，一次被left）。
- **空间复杂度**：O(1)，使用固定大小的数组存储字符计数。

## 关键点

1. **字符计数**：使用数组而非哈希表，效率更高
2. **valid变量**：避免每次都比较整个计数数组
3. **先扩展后收缩**：保证窗口内信息完整
4. **边界条件**：注意处理空串、无解等情况
