# LeetCode 567: 字符串的排列

## 题目描述

给你两个字符串 `s1` 和 `s2`，写一个函数来判断 `s2` 是否包含 `s1` 的排列。

如果是，返回 `true`；否则，返回 `false`。

换句话说，`s1` 的排列之一是 `s2` 的子串。

## 示例

**示例 1：**
```
输入：s1 = "ab", s2 = "eidbaooo"
输出：true
解释：s2 包含 s1 的排列 "ba"
```

**示例 2：**
```
输入：s1 = "ab", s2 = "eidboaoo"
输出：false
```

## 形象化理解

### 🎯 问题本质

想象你在玩拼图游戏：
- 你有一块完整的拼图（s1）
- 想看看另一堆拼图（s2）中是否存在完全相同的拼图块组合，只是顺序可能不同

### 🔍 算法图解

```
s1 = "ab" → 需要找：一个 'a' 和一个 'b' 的组合

s2 = "e i d b a o o o"
            ↑ ↑
        找到了！'b' 和 'a' 相邻

滑动窗口视角：
┌───┐
│ e │ i d b a o o o   窗口：ei → 不匹配
└───┘
  ┌───┐
│ e i │ d b a o o o   窗口：id → 不匹配
  └───┘
    ┌───┐
│ e i d │ b a o o o   窗口：db → 不匹配
    └───┘
      ┌───┐
│ e i d b │ a o o o   窗口：ba → 匹配！✓
      └───┘
```

## 解题思路

### 方法：固定窗口滑动

这道题的关键特点是：**窗口大小固定**！

1. **窗口大小固定为 s1 的长度**
   - 因为 s1 的任意排列长度都等于 s1 的长度
   - 我们只需要在 s2 上用固定大小的窗口滑动

2. **字符计数比较**
   - 统计 s1 中每个字符的出现次数
   - 对于每个窗口，统计窗口内字符的出现次数
   - 如果两个计数完全相同，说明找到了排列

3. **增量更新优化**
   - 窗口滑动时，只需要更新两个字符的计数
   - 新进入窗口的字符：计数 +1
   - 离开窗口的字符：计数 -1

### 代码模板

```cpp
bool checkInclusion(string s1, string s2) {
    int n1 = s1.size(), n2 = s2.size();
    if (n1 > n2) return false;
    
    // 统计 s1 的字符计数
    vector<int> count1(26, 0);
    for (char c : s1) count1[c - 'a']++;
    
    // 初始化窗口
    vector<int> count2(26, 0);
    for (int i = 0; i < n1; i++) {
        count2[s2[i] - 'a']++;
    }
    
    if (count1 == count2) return true;
    
    // 滑动窗口
    for (int i = n1; i < n2; i++) {
        count2[s2[i] - 'a']++;           // 加入新字符
        count2[s2[i - n1] - 'a']--;      // 移除旧字符
        if (count1 == count2) return true;
    }
    
    return false;
}
```

## 复杂度分析

- **时间复杂度**：O(n)，其中 n 是 s2 的长度
  - 初始化窗口：O(n1)
  - 滑动窗口：O(n2 - n1)
  - 数组比较：O(26) = O(1)
  
- **空间复杂度**：O(1)
  - 使用固定大小的数组（26个元素）
